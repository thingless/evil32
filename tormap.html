Tor's Map of the Internet

The area of each website corresponds to the amout of traffic received from the Tor network.

"Who uses Tor? Just Criminals?"

I have supported the Tor project for years and a friend recently challenged if I should.
This map is the result of my friend's challenge.

Q&A

How was this data collected?

Domain lookups were recorded from several exit relays for a couple months.

Isn't it against the spirit of Tor to record connection data?

Kinda, but only domain lookups were recorded which my ISP probably already does :(

How accurate is the data you collected?

It's sampled from only a few exit relays, and DNS data isn't exactly clean. It's a rough estimate.

Wait! What about hidden services? That area of the map is all foggy!

Yea, I don't know of a way to collect usage info on hidden services :)
